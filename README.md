# Zero Trust Security Model

A Flask-based Zero Trust Security simulation that dynamically evaluates user trust based on behavior, session activity, and risk factors.

## Features

* User Authentication Portal
* Dynamic Trust Score Calculation
* Administrative Monitoring Dashboard
* Session Risk Analysis
* Sensitive Resource Tracking
* Automatic Logout for Low-Trust Users
* Real-Time Trust Evaluation Engine

## Tech Stack

* Python
* Flask
* HTML
* CSS
* Jinja2 Templates

## Project Structure

```text
zero-trust-demo/
│
├── app.py
├── trust_engine.py
│
└── templates/
    ├── login.html
    ├── dashboard.html
    └── admin.html
```

## Running the Project

```bash
pip install flask
python app.py
```

Open your browser and visit:

```text
http://localhost:5000
```
